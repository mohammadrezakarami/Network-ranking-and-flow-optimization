# Network Ranking and Flow Optimization

This repository contains a notebook and supporting MATLAB data files for experiments related to network ranking and flow optimization.

## Repository Contents

- `code.ipynb`: main notebook for the analysis and implementation.
- `report.pdf`: project report.
- `capacities.mat`, `flow.mat`, `traffic.mat`, `traveltime.mat`: supporting data files.

## Topics Covered

- network flow optimization
- ranking and routing analysis
- notebook-based experimentation
- hybrid Python and MATLAB-style data usage

## Getting Started

1. Clone the repository.
2. Open the notebook in Jupyter.
3. Ensure the notebook environment can load MATLAB `.mat` files.
4. Use the report for additional context and interpretation.

## Notes

This repository is intended as an academic project repository combining code, data, and written report material.
